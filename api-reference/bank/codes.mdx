---
title: "Códigos de bancos"
api: "GET https://api.venepagos.com.ve/api/v1/bank/codes"
description: "Obtiene la lista de códigos de bancos venezolanos."
---

<Note>
  Este endpoint requiere autenticación mediante Bearer token.
</Note>

### Respuesta

<ResponseField name="code" type="string">
  Código del banco (4 dígitos).
</ResponseField>

<ResponseField name="name" type="string">
  Nombre del banco.
</ResponseField>
