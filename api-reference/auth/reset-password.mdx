---
title: "Restablecer contraseña"
api: "POST /auth/reset-password"
description: "Restablece la contraseña del usuario usando el código OTP recibido por correo."
---

<ParamField body="email" type="string" required>
  Correo electrónico de la cuenta.
</ParamField>

<ParamField body="otp" type="string" required>
  Código OTP de 6 dígitos recibido por correo electrónico.
</ParamField>

<ParamField body="newPassword" type="string" required>
  Nueva contraseña. Debe tener al menos 8 caracteres.
</ParamField>

### Respuesta

<ResponseField name="message" type="string">
  Mensaje de confirmación del restablecimiento.
</ResponseField>

<ResponseExample>
```json
{
  "message": "Contraseña restablecida exitosamente."
}
```
</ResponseExample>
