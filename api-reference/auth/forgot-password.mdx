api: "POST /auth/forgot-password"
description: "Envía un código OTP al correo electrónico del usuario para restablecer la contraseña."
---

<ParamField body="email" type="string" required>
  Correo electrónico de la cuenta a recuperar.
</ParamField>

### Respuesta

<ResponseField name="message" type="string">
  Mensaje de confirmación del envío del código.
</ResponseField>

<ResponseExample>
```json
{
  "message": "Se ha enviado un código de recuperación a tu correo electrónico."
}
```
</ResponseExample>
