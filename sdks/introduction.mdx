VenePagos ofrece librerías oficiales para facilitar la integración de nuestra API en tu aplicación. Todos nuestros SDKs están diseñados para ser intuitivos, seguros y consistentes entre sí.

### Características comunes

Todos los SDKs oficiales de VenePagos incluyen:

- **Autenticación simplificada**: Manejo automático de Bearer tokens.
- **Entornos**: Soporte nativo para `live` y `test` (sandbox).
- **Manejo de errores**: Excepciones descriptivas para facilitar la depuración.
- **Tipado fuerte**: Interfaces y modelos de datos completos.
- **Validación**: Validación previa de parámetros comunes como RIF y cuentas bancarias.

<Tip>
  Si prefieres no usar un SDK, siempre puedes consumir nuestra [API REST](/api-reference/introduction) directamente.
</Tip>
