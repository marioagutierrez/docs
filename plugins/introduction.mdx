Ofrecemos plugins gratuitos para las plataformas de e-commerce más utilizadas. Estos plugins permiten aceptar pagos de forma nativa sin necesidad de escribir código.

### Plataformas soportadas

<CardGroup cols={2}>
  <Card title="WooCommerce" icon="wordpress" href="/plugins/woocommerce">
    Acepta pagos móviles, transferencias y tarjetas en tu sitio WordPress.
  </Card>
  <Card title="Shopify" icon="shopify" href="/plugins/shopify">
    Integración oficial para el checkout de Shopify mediante App personalizada.
  </Card>
</CardGroup>

### Ventajas de usar plugins

- **Instalación rápida**: Configuración en menos de 5 minutos.
- **Sincronización de pedidos**: El estado del pedido se actualiza automáticamente al confirmarse el pago.
- **Seguridad**: Los datos sensibles no tocan tu servidor.
- **Soporte nativo**: Integración perfecta con el flujo de checkout estándar de cada plataforma.

<Note>
  Todos los plugins requieren una cuenta activa en VenePagos y una API Key configurada.
</Note>
