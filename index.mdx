## Bienvenido a VenePagos

VenePagos es la plataforma de pagos diseñada para el comercio digital en Venezuela. Procesa pagos en línea con seguridad bancaria, integración por API y liquidación directa a tu cuenta.

## Métodos de pago soportados

- Links de pago personalizados (compartibles por WhatsApp, email, redes sociales)
- Transferencias bancarias (SIMF)
- Pagos móviles C2P (pago por celular)
- Integración directa por API RESTful
